This folder contains D-Type Engines in the form of 32-bit shared libraries for Linux (Intel 386)

* libdtypestd.so.8.2.1 - D-Type Standard Engine
* libdtypepwr.so.8.2.1 - D-Type Power Engine
* libdtypetxt.so.8.2.1 - D-Type Unicode Text Module
* libdtypelay.so.8.2.1 - D-Type Text Layout Extension (using HarfBuzz)
* libdtypeaux.so.8.2.1 - D-Type D-Type External Format Plugin
